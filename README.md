# iMenuRNative

Android Studio yükleyip sanal bir Android cihaz yapılması gerekiyor, Android 12 (API 31) sürümü , intel x86 Atom_64 system image, Google APIs intel x86 Atom_64,
SDK Tools kısmındanda SDK Build 31.0.0, NDK Son sürüm, SDK Command line tools son sürüm, Android Emulator, Android SDK Platform-tools, Google Play Services, Intel x86
emulator accelerator(HAXM) kurulması gerekiyor.

Sanal Makine kurulduktan sonra;

npm install 

npx react-native run-android

Sırası ile yapıldığında bir sorun kalmamış olması lazım.
